What is Dozera?

Dozera is a personal dashboard platform where every user can view their realtime activity data in one secure place. After logging in, the website connects directly to Firebase Realtime Database, and all data related to that account — whatever has been saved, such as photos, notes, activity logs, or any other field — is automatically displayed here.

Every user is given two special identifiers at login — a PS Code that is shown in full, and a PX Code that stays partially hidden for privacy. Both are 15-character alphanumeric codes that work like a unique API-key-style identity, and are stored only inside that user's own profile.

Dozera's design is kept simple and clean — without any unnecessary visual complexity — so the user can focus only on their data. The website supports both zoom-in and zoom-out, so it's comfortable to read on any screen size.
